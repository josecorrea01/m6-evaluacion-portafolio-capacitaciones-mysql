Sistema de Capacitaciones Internas

Este sistema permite gestionar cursos de capacitación para los colaboradores de la empresa, administrando instructores, cupos e inscripciones. Utiliza Spring Boot, Spring Security, Spring Data JPA, Thymeleaf y MySQL.

Ir al panel de administración Ver cursos como empleado

Resumen de funcionalidades

  • • Login con roles ADMIN y EMPLEADO.
  • • Panel ADMIN para gestionar cursos.
  • • Panel EMPLEADO para ver cursos e inscribirse.
  • • API REST para listar cursos e inscripciones.
  • • Persistencia con Spring Data JPA + MySQL.

Rol ADMIN

Usuario responsable de la creación y administración de cursos, asignación de instructores y revisión de la oferta de capacitaciones internas.

  • Gestionar cursos y cupos.
  • Asignar instructores.
  • Revisar cursos activos/inactivos.
Ruta: /admin/cursos

Rol EMPLEADO

Colaborador que puede revisar los cursos publicados, conocer los detalles de cada capacitación e inscribirse en las que estén disponibles.

  • Consultar cursos disponibles.
  • Revisar fechas y cupos.
  • Realizar inscripciones.
Ruta: /empleado/cursos
